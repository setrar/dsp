   {\color{Green} \mathbf{ Convergence \space of \space Fourier \space series } } \\

```math
\begin{gather}
   \\

    \\
   {\color{Yellow} \text{ Finite energy over one period } } \\
   {\color{Yellow} \text{ (Absolutely integrable } } \\
\end{gather}
```

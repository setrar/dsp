```math

   {\color{Purple} \mathbf{ Delta \space Functions } }
\end{gather}
```

```math
\begin{gather}
    x[-1]\delta[n + 1] =
      \begin{cases}
        x[ -1 ],   & \quad n = -1 \\
        0,         & \quad n \neq -1
      \end{cases} \\
    \\
    x[0]\delta[n ] =
      \begin{cases}
        x[ 0 ],   & \quad n = 0\\
        0,        & \quad n \neq 0
      \end{cases} \\
    \\
    x[1]\delta[n + 1] =
      \begin{cases}
        x[ 1 ],   & \quad n = 1 \\
        0,        & \quad n \neq 1
      \end{cases} \\
    \\
   {\color{Cerulean} \text{ Sifting Property } }
\end{gather}
```
